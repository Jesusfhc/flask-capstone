About Me

This is what I do.

Este es un blog que cree utilizando un framework de python llamado Flask, con el cual tomo plantillas o archivos que contienen el html, css y javascript. Y les doy logica, asi como funcionalidad a la aplicacion y su despliegue web

Esto consiste en un simple blog personal, donde esta el administrador (mi usuario) que puede crear, editar o eliminar las publicaciones, asi como usuarios que cualquiera puede crear para comentar estas publicaciones

El contenido que se vera en este blog son opiniones mias acerca de distintos ambitos como albums de musica, animes en general, manga o cualquier tema
